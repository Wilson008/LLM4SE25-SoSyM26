|---------------|--------------|--------|

| Top-level `event UpdateNameDomainEvent` |  **Removed** | `IsisEvent` is no longer allowed at `IsisEntity` top-level (only inside `property`, `collection`, `action` blocks). Instead, event association is handled via `@Action(domainEvent = ...)` annotation - preserved. |
